CSS Sprites是什么？

CSS雪碧图、CSS精灵，将一个页面涉及到的所有零星图片都包含到一张大图中,再通过利用CSS的background-image，background- repeat，background-position的组合进行背景定位.background-position可以用数字精确的定位出背景图片的位置

或者直接这样写：

它有什么作用？

为了分离图片，设置或检索对象的背景图像位置，最终Make Fewer HTTP Requests.

解决什么问题？

客户端每显示一张图片都会向服务器发送请求。所以，图片越多请求次数越多，造成延迟的可能性也就越大.
