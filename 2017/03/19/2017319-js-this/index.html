为什么用this

先看段没有使用this的代码，没用this就需要给sleep()显示传入一个对象引用.


随着使用模式越来越复杂，显示传递对象引用会让代码变得混乱，因此可以将代码设计得更简洁并且易于复用.

this是什么

this是在函数被调用时发生的绑定，它指向什么完全取决于函数在哪里被调用
